# @veloss/vite-t3-env

vite 환경에서 환경변수를 t3-env와 쉽게 연동하고 타입을 자동으로 만들어주기 위한 vite plugins 입니다.

## 🚀 Quick Start

Install:

```bash
# npm
npm install @veloss/vite-t3-env

# yarn
yarn add @veloss/vite-t3-env

# pnpm
pnpm add @veloss/vite-t3-env
```

Import:

```ts
// ESM / Typescript
import { t3EnvPlugin } from "@veloss/vite-t3-env";
```
